Menu Design

This project is where I designed a menu for Nusara, a Japanese–Asian fusion restaurant. The aim was to create a premium yet culturally rooted visual experience. For the menu, I used a tall, elegant layout with a soft pink base and traditional Japanese patterns. The typography is minimal and modern to keep it clean and readable. There are various sections with clear icons for vegetarian dishes.

Role: Graphic Design
Tools: Adobe Illustrator, Photoshop
